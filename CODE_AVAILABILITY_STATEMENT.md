# Code availability statement

The analysis code is available in the GitHub repository:

https://github.com/afa-cloud/escc-spatial-bulk-validation-framework-code

The manuscript-submission clean-text code snapshot is release tag `v1.0.2-submission-cleantext` at commit `b037967848d066db05ddd3c23c664e73034cbcc9`. It contains public-data analysis code and configuration files only. The release is archived on Zenodo at https://doi.org/10.5281/zenodo.19827735.

A copy of the submitted code package is also provided as S2 Code with the manuscript.
