# v1.0.1-submission-cleantext

This release contains a clean-text public analysis code package for peer review.

Changes relative to `v1.0.0-submission-sanitized`:

- Replaced internal workflow terminology in public scripts and package metadata with neutral reproducibility terminology.
- Replaced local tool-specific HTTP user-agent labels with project-neutral labels.
- Kept the code package limited to public-data analysis scripts, configuration files, manifests and processed supporting tables.

The release does not contain manuscript drafts, administrative submission files, local quality-control utilities, full-text PDFs, controlled-access data, API keys or unpublished human-subject data.
