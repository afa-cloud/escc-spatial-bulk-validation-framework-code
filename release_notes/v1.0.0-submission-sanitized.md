# v1.0.0-submission-sanitized

This release contains the sanitized public analysis code package for peer review.

Changes relative to the earlier submission archive:

- Removed manuscript drafting, DOCX generation, cover-letter generation, and internal audit-rendering scripts from the public code surface.
- Added `scripts/public_data_helpers.py` so analysis scripts do not depend on unpublished local workflow modules.
- Sanitized data manifests to avoid local absolute paths.
- Updated code-availability metadata to point to the sanitized GitHub release and submitted S2 Code package.

The release contains public-data analysis code only. It does not contain manuscript drafts, reviewer notes, full-text PDFs, controlled-access data, API keys, or unpublished human-subject data.
